There is always a way to improve something. You can take a look at Generators for Python.

Ok. How can I use it?

You can simply replace code

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    result = []
    for item in data:
        if data.count(item) > 1:
            result.append(item)

    return result

with this:

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    return (a for a in data if data.count(a) > 1)

It is shorter and even easier to read.
