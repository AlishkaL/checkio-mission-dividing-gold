Le capitaine Œil rouge et ses marins ont trouvé un coffre contenant un grand nombre de pièces d'or identiques. Pour partager les pièces, tout le monde est d'accord avec la suggestion du capitaine:

  • chaque marin, sauf le capitaine, doit recevoir exactement le même nombre de pièces;
  • le capitaine doit recevoir deux fois plus de pièces qu'un marin.
